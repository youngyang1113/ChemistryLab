# 🧪 AI Chemistry Lab（改进版）

一个面向高中教学的虚拟化学实验平台，支持拖拽试剂、超真实动画、温度/pH实时显示、教师控制台、任务模式、日志导出、撤销功能。

## ✨ 功能特点（全新升级）

- **拖拽试剂到烧杯**：自动配对反应，支持 **48 种** 常见试剂。
- **超仿真动画系统**（全面升级）：
  - 发热反应：多层火焰 + 真实热浪 + 烟雾
  - 沉淀反应：沉淀生长 + 多样颗粒沉降
  - 产气反应：密集气泡 + 水平摆动 + 渐隐
  - 置换/氧化还原：金属光泽 + 平滑颜色变化
  - 液体颜色动态变化 + **真实液位上升**
- **温度计 & pH 计**：实时显示温度与酸碱度。
- **智能提示**：不反应时智能建议。
- **教师控制台**：实时记录 + 清空 + 一键导出 CSV。
- **任务模式**：支持沉淀、产气、置换、中和、氧化还原五种任务。
- **AI讲解 + 撤销上一步 + 简单声音反馈**。
- **玻璃拟态 UI**：现代毛玻璃效果。
